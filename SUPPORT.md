# Support

## Where to get help

- **Documentation:** see the `docs/` directory and each package `README.md`.
- **Questions and discussion:** open a [GitHub Discussion](../../discussions).
- **Bug reports:** open a [GitHub Issue](../../issues) using the bug-report template.
- **Feature requests:** open a [GitHub Issue](../../issues) using the feature-request template.

## What is not supported here

This repository is a public demo. Production support SLAs are not offered.
